Desarrollo de Magic Books

Listado de algunas de las herramientas que se utilizaron para el desarrollo del sistema...

Visual Studio

Visual Studio fue el IDE que se utilizo para el desarrollo del sistema Magic Books siendo el lenguaje de programación C#.

PHPMyAdmin

PHPMyAdmin fue el gestor de base de datos que se utilizo, para realizar la base de datos que se utiliza en el sistema, tambien se le hizo matenimiento durante la creación del sistema e incluir que el tipo de base de datos que se manejo fue SQL.

Sublime Text

Sublime Text es el un editor de texto, en donde elaboramos la pagina web, con todos sus elementos, las paginas web incluye el codigo fuente HTML, el estilo de la pagina que es CSS y para el dinamismo de la pagina utilizamos JavaScript.

Balsamiq Mockups

Con Balsamiq Mockups se diseño lo que es el boceto de la pagina web y de los formularios del sistema, esta herramienta nos facilito mucho lo que es el diseño para llevar todo en orden y dar forma a todos los elementos que el sistema utiliza.

Magic Books

El proyecto Magic Books es un programa el cual será usado en una biblioteca, respondiendo a los problemas de difícil organización de todos los libros que se encuentran y la dificultad al pasar a retirar un libro o al devolver un libro y así con el programa solventar esta problemática.

El programa Magic Books solventará la necesidad de una biblioteca, para la organización y agrupación de esta según la categoría de cada libro y la necesidad del público, para facilitar la entrega y recibimiento de cada libro, este contará con su propia organización de género y a partir de esta dependería el orden alfabético

Quienes Somos

En este apartado encontraras nuestra Misión, Visión y Objetivo.

Misión

El Sistema Magic Books (Management System) tiene como misión facilitar a la comunidad estudiantil recursos y servicios informativos de calidad, con eficiencia, oportunidad y confiabilidad tecnológica.

Visión

Facilitar recursos y servicios de información con pertinencia académica y de calidad; para fomentar una cultura de gestión de la información entre la comunidad educativa y propicie un entorno favorable para el aprendizaje y la generación de nuevos conocimientos.

Objetivo

Fortalecer y consolidar el Sistema Institucional Bibliotecario mediante la disposición y uso de infraestructura y tecnología informática en red.
